# **App Name**: CertSecure Demo

## Core Features:

- Certificate Upload: Allow users to upload certificate files to Firebase Storage.
- Certificate Verification: Verify the uploaded certificate using OpenSSL.
- User Authentication: Enable user authentication via Email/Password using Firebase Authentication.
- Access Control: Restrict access to certain certificates using Firebase Authentication roles and Firestore rules.
- Results Display: Display the results of certificate verification with OpenSSL to the user after performing verification.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to symbolize trust and security.
- Background color: Light pale green (#F1F8E9), offering a soft and clean backdrop.
- Accent color: Teal (#008080) to provide contrast, and complement the green hues.
- Font pairing: 'Inter' (sans-serif) for body text and 'Literata' (serif) for headlines, which together offers an elegant, modern look.
- Use security-themed icons to symbolize aspects such as verification and encryption.
- Use a single column layout to work well on mobile devices, and a multiple column layout on larger screens.
- Subtle animations should confirm success, show progression, and provide hints or feedback for user input
